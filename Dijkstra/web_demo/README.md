# Dijkstra Web Demo

Interactive React visualization of Dijkstra's Algorithm on the India Road Network.

## Features
- Select any source and destination city
- See shortest path highlighted with total distance
- Step-by-step algorithm walkthrough
- All city distances shown in one view

## How to Run
```bash
npx create-react-app dijkstra-demo
cd dijkstra-demo
cp ../App.jsx src/App.js
npm start
```

Or paste App.jsx into https://codesandbox.io for instant preview.

> This is a bonus visual demo. The main Python implementation is in ucs_dijkstra.py
